# AI资产管理页面需求规格说明书

## 一、页面概述

### 1.1 页面定位
AI资产管理页面管理AI模型、数据集和容器镜像，为任务中心提供资源支撑，是AI工作流的基础资源管理模块。

### 1.2 页面功能
- 管理模型仓库（预置模型和自定义模型）
- 管理数据集
- 管理镜像仓库
- 提供模型服务市场入口

## 八、业务规则

### 8.1 模型使用规则
- 预置模型可直接用于创建任务
- 自定义模型需先上传验证后可用
- 模型可部署为推理服务

### 8.2 数据集使用规则
- 数据集可挂载到训练任务
- 数据集可挂载到推理服务
- 数据格式需符合任务要求

### 8.3 镜像使用规则
- 镜像用于定义任务运行环境
- 预置镜像可直接选择使用
- 自定义镜像需推送至镜像仓库

## 九、交互流程

### 9.1 查看模型仓库

```
进入AI资产管理页面
        │
        ↓
加载模型仓库标签页
        │
        ↓
显示预置模型列表
        │
        ├──→ 用户切换到自定义模型 → 显示私有模型
        │
        ├──→ 用户点击模型卡片 → 查看模型详情
        │
        └──→ 用户点击部署服务 → 创建推理服务
```

### 9.2 上传数据集流程

```
切换到数据集标签页
        │
        ↓
点击上传数据集
        │
        ↓
选择本地文件
        │
        ↓
填写数据集信息
        │
        ↓
上传文件
        │
        ↓
数据集入库 → 刷新列表
```

### 9.3 使用资源创建任务

```
在AI资产管理页面
        │
        ↓
选择模型/数据集/镜像
        │
        ↓
点击"用于创建任务"
        │
        ↓
跳转到任务中心 → 资源已预选
```
