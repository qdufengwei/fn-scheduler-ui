# 租户资源页面需求规格说明书

## 一、页面概述

### 1.1 页面定位
租户资源页面是集群资源模块下的核心页面，用于管理和展示GPU配额在各租户间的分配情况，是资源分配的核心管理页面。通过白名单机制将GPU资源分配给租户。

### 1.2 页面功能
- 展示集群GPU资源统计概览
- 展示各租户的配额分配明细
- 支持按租户筛选配额记录
- 支持缩减资源和释放资源操作

## 三、资源统计卡片

### 3.1 GPU总资源

| 属性 | 说明 |
|------|------|
| 统计项 | GPU总资源 |
| 数值 | 集群总GPU卡数 |
| 单位 | 卡 |
| 说明 | vGPU/MIG已转换为等效卡数 |
| 子项 | 整卡资源、MIG资源、vGPU资源 |

### 3.2 已分配（白名单）

| 属性 | 说明 |
|------|------|
| 统计项 | 已分配(白名单) |
| 数值 | 管理员分配给租户的GPU卡数 |
| 单位 | 卡 |
| 说明 | 包含逻辑GPU等效分配 |
| 子项 | 整卡资源、MIG资源、vGPU资源 |

### 3.3 已出售（用户购买）

| 属性 | 说明 |
|------|------|
| 统计项 | 已出售(用户购买) |
| 数值 | 用户通过购买获得的GPU卡数 |
| 单位 | 卡 |
| 说明 | 包含逻辑GPU等效售出 |
| 子项 | 整卡资源、MIG资源、vGPU资源 |

### 3.4 剩余可用配额

| 属性 | 说明 |
|------|------|
| 统计项 | 剩余可用配额 |
| 数值 | 可分配的剩余GPU卡数 |
| 单位 | 卡 |
| 说明 | 显示总体可用率百分比 |
| 子项 | 整卡资源、MIG资源、vGPU资源 |
| 计算方式 | GPU总资源 - 已分配 - 已出售 |

## 五、配额列表字段

### 5.1 字段明细

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
|------|----------|----------|----------|--------|
| 1 | 租户/别名 | 文本 | 租户名称，无租户显示"-"，有别名显示别名 | test-0415 |

## 八、业务规则

### 8.1 配额计算规则
- **GPU总资源** = 所有已上架节点的GPU总数
- **已分配** = 白名单分配的GPU总数
- **已出售** = 用户购买的GPU总数
- **剩余可用** = 总资源 - 已分配 - 已出售
- **可用率** = 剩余可用 / 总资源 × 100%

### 8.2 已用/总计说明
- **已用**：租户当前正在使用的GPU数量
- **总计**：租户拥有的GPU配额总数
- 当已用 > 总计时，说明资源超配（需要关注）

### 8.3 存储配额说明
- 存储配额独立于GPU配额
- 格式：已用存储 / 总存储配额

## 九、交互流程

### 9.1 查看配额流程

```
进入租户资源页面
        │
        ↓
加载资源统计卡片
        │
        ↓
加载配额列表
        │
        ├──→ 用户选择租户筛选 → 过滤列表
        │
        └──→ 用户点击操作按钮 → 执行对应操作
```

### 9.2 缩减资源流程

```
点击"缩减资源"
        │
        ↓
弹出缩减资源弹窗
        │
        ↓
输入要缩减的数量
        │
        ↓
确认缩减
        │
        ↓
更新租户配额 → 刷新列表
```

### 9.3 释放资源流程

```
点击"释放资源"
        │
        ↓
弹出确认对话框
        │
        ↓
确认释放
        │
        ↓
校验是否有使用中的资源
        │
   ┌────┴────┐
   ↓         ↓
  有        无
   │         │
   ↓         ↓
提示警告   释放资源 → 更新统计 → 刷新列表
```
