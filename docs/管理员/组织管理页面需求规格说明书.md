# 组织管理页面需求规格说明书

## 一、页面概述

### 1.1 页面定位
组织管理页面是平台的管理入口，用于创建和管理租户及用户，是平台权限和资源分配的基础。

### 1.2 页面功能
- 管理租户（创建、编辑、删除）
- 管理用户（创建、编辑、禁用、删除）
- 设置租户折扣
- 分配租户管理员

## 七、业务规则

### 7.1 租户规则
- 租户名称必须唯一
- 租户创建时必须分配租户管理员
- 删除租户前必须释放所有资源

### 7.2 用户规则
- 用户名必须唯一
- 用户必须归属于某个租户
- 租户管理员不能删除（需先转移权限）

### 7.3 权限规则
- 系统管理员可管理所有租户和用户
- 租户管理员可管理本租户内的用户
- 普通用户只能管理自己的资源

## 八、交互流程

### 8.1 创建租户流程

```
点击"添加租户/租户管理员"
        │
        ↓
弹出创建表单
        │
        ↓
填写租户信息和管理员信息
        │
        ↓
点击"确认"
        │
        ↓
创建租户 → 创建租户管理员 → 刷新列表
```

### 8.2 创建用户流程

```
切换到用户管理标签页
        │
        ↓
点击"添加用户"
        │
        ↓
弹出创建表单
        │
        ↓
填写用户信息
        │
        ↓
选择所属租户和角色
        │
        ↓
点击"确认" → 创建用户 → 刷新列表
```

### 8.3 删除租户流程

```
点击"删除"按钮
        │
        ↓
检查租户关联资源
        │
   ┌────┴────┐
   ↓         ↓
  有资源    无资源
   │         │
   ↓         ↓
提示警告   弹出确认对话框
需先释放           │
                   ↓
              确认删除 → 删除租户 → 刷新列表
```

### 8.4 重置密码流程

```
点击"重置密码"按钮
        │
        ↓
弹出重置密码对话框
        │
        ↓
输入新密码
        │
        ↓
点击"确认" → 密码重置成功
```
