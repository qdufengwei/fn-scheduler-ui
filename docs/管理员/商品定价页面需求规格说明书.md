# 商品定价页面需求规格说明书

## 一、页面概述

### 1.1 页面定位

商品定价页面是计费管理模块下的子页面，用于配置GPU和存储资源的计费价格，包括基础价格和包年包月折扣价格。

### 1.2 页面功能

- 展示所有资源类型的定价配置
- 支持编辑资源定价
- 支持设置基础价格（按需计费）
- 支持设置包月/包年折扣价

## 三、列表字段定义

### 3.1 字段明细

| 序号 | 字段名称   | 字段类型 | 字段说明              | 示例值                |
| ---- | ---------- | -------- | --------------------- | --------------------- |
| 1    | 资源类型   | 文本     | GPU型号或存储类型名称 | NVIDIA-GPU-HBM2E-80GB |
| 2    | 基础价格   | 货币     | 按需计费的基础价格    | 1.00 (¥ / 卡 / 时)    |
| 3    | 包月折扣价 | 货币     | 包月购买的折扣价格    | -                     |
| 4    | 包年折扣价 | 货币     | 包年购买的折扣价格    | -                     |
| 5    | 操作       | 按钮     | 编辑定价              | [编辑]                |

### 3.2 资源类型说明

| 资源类型              | 类型分类    | 说明              |
| --------------------- | ----------- | ----------------- |
| NVIDIA-GPU-HBM2E-80GB | GPU计算资源 | GPU型号定价       |
| rook-cephfs           | 分布式存储  | CephFS存储定价    |
| nfs-client            | NFS存储     | NFS客户端存储定价 |
| nfs-storage           | NFS存储     | NFS存储定价       |
| standard              | 标准存储    | 标准存储类型定价  |
| local-path            | 本地存储    | 本地路径存储定价  |

### 3.3 价格格式说明

- 基础价格格式：`X.XX (¥ / 卡 / 时)` 或 `X.XX (¥ / GB / 时)`
- 包月/包年折扣价：若无配置显示"-"

## 四、编辑定价弹窗

### 4.1 弹窗标题

### 4.2 表单字段

| 序号 | 字段名称 | 是否必填 | 控件类型 | 说明 |

| 2 | 包月折扣价 | 否 | 数字输入框 + 单位 | 包月购买的价格 |
| 3 | 包年折扣价 | 否 | 数字输入框 + 单位 | 包年购买的价格 |

| 按钮 | 功能               |
| ---- | ------------------ |
| 确认 | 保存定价修改       |
| 取消 | 取消编辑，关闭弹窗 |

## 六、业务规则

### 6.1 定价规则

- **基础价格**：按需计费的默认单价
- **包月折扣价**：购买一个月的优惠价格
- **包年折扣价**：购买一年的优惠价格（通常比包月更优惠）

### 6.2 价格单位

- GPU资源：¥ / 卡 / 时（每卡每小时价格）
- 存储资源：¥ / GB / 时 或 ¥ / GB / 月

### 6.3 定价生效规则

- 修改定价后立即生效
- 对新创建的订单应用新价格
- 已创建的订单按原价格执行

## 七、交互流程

### 7.1 查看定价列表

```
进入商品定价页面
        │
        ↓
加载所有资源类型的定价配置
        │
        ↓
显示定价列表
```

### 7.2 编辑定价流程

```
点击"编辑"按钮
        │
        ↓
弹出编辑定价弹窗
        │
        ↓
回显当前定价值
        │
        ↓
用户修改价格
        │
        ↓
点击"确认"
        │
        ↓
校验输入值
        │
   ┌────┴────┐
   ↓         ↓
  失败      成功
   │         │
   ↓         ↓
显示错误   保存定价 → 提示成功 → 刷新列表
```
