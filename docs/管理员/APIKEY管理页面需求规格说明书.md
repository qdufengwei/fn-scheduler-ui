# API KEY管理页面需求规格说明书

## 一、页面概述

### 1.1 页面定位
API KEY管理页面用于管理调用平台API的密钥，主要用于推理服务的外部调用鉴权，是API访问的安全入口。

### 1.2 页面功能
- 展示API KEY统计信息
- 展示API KEY列表
- 创建新的API KEY
- 编辑API KEY权限
- 禁用/启用API KEY
- 查看API调用示例

## 五、API KEY列表字段

### 5.1 字段明细

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
|------|----------|----------|----------|--------|
| 1 | 名称/ID | 文本 | API KEY名称和密钥标识 | test / MhcrstnXjSo |
| 2 | 权限 | 标签列表 | API KEY的权限配置 | 推理服务：读取、推理服务：写入/执行 |
| 3 | 状态 | 标签 | API KEY状态：运行、已禁用 | 运行 |
| 4 | 租户/别名 | 文本 | 所属租户 | test01 |

| 权限 | 说明 |
|------|------|
| 推理服务：读取 | 可读取推理服务信息和结果 |
| 推理服务：写入/执行 | 可创建和执行推理请求 |

| 状态 | 说明 |
|------|------|
| 运行 | API KEY可用，可正常调用 |

## 十、业务规则

### 10.1 API KEY生成规则
- KEY ID自动生成，不可修改
- KEY格式为随机字符串（如：MhcrstnXjSo）
- 创建后立即可用

### 10.2 权限控制规则
- 不同权限对应不同的API访问范围
- 读取权限：可查询推理服务状态和结果
- 写入/执行权限：可提交推理请求

### 10.3 禁用规则
- 禁用后KEY立即失效
- 禁用的KEY可重新启用
- 有绑定推理任务的KEY禁用时需提示

## 十一、交互流程

### 11.1 创建API KEY流程

```
点击"创建API KEY"
        │
        ↓
弹出创建表单
        │
        ↓
填写名称和权限
        │
        ↓
选择所属租户
        │
        ↓
点击"确认"
        │
        ↓
生成API KEY → 显示KEY信息 → 刷新列表
```

### 11.2 禁用API KEY流程

```
点击"禁用"按钮
        │
        ↓
检查绑定推理任务数量
        │
   ┌────┴────┐
   ↓         ↓
  有绑定    无绑定
   │         │
   ↓         ↓
提示警告   直接禁用 → 状态变为"已禁用"
确认后禁用
```

### 11.3 查看示例流程

```
点击"示例"按钮
        │
        ↓
展示API调用代码示例
        │
        ↓
用户可复制示例代码
```
