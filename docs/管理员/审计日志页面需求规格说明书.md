# 审计日志页面需求规格说明书

## 一、页面概述

### 1.1 页面定位
审计日志页面是集群资源模块下的子页面，记录平台所有操作行为，用于安全审计、问题追溯和合规检查。

### 1.2 页面功能
- 展示所有操作日志记录
- 支持按租户筛选
- 支持按用户筛选
- 支持时间范围查询
- 支持日志下载导出

---

## 二、页面布局

### 2.1 整体结构

```
┌─────────────────────────────────────────────────────────────────────┐
│                         顶部导航栏                                   │
├─────────────────────────────────────────────────────────────────────┤
│  审计日志                                                           │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  筛选：[请选择租户▼]  [请选择用户▼]                         │   │
│  │  时间范围：[开始日期] 至 [结束日期]        [日志下载]        │   │
│  └─────────────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                     日志列表表格                             │   │
│  │  操作者 | 租户 | 操作行为 | 请求结果 | 操作时间 | 请求方法   │   │
│  └─────────────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  分页控件：上一页 | 1 2 3 ... 52 | 下一页 | 每页条数 | 跳转   │   │
│  └─────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────┘
```

---

## 三、筛选功能

### 3.1 筛选项

| 筛选项 | 控件类型 | 说明 |
|--------|----------|------|
| 租户选择 | 下拉选择框 | 按租户筛选操作记录 |
| 用户选择 | 下拉选择框 | 按操作用户筛选 |
| 开始日期 | 日期时间选择器 | 查询的起始时间 |
| 结束日期 | 日期时间选择器 | 查询的结束时间 |

### 3.2 筛选逻辑
- 支持多条件组合筛选
- 租户和用户可同时选择
- 时间范围为必选条件

---

## 四、功能按钮

### 4.1 日志下载

| 属性 | 说明 |
|------|------|
| 按钮 | 日志下载 |
| 功能 | 导出筛选后的审计日志文件 |
| 格式 | CSV或Excel格式 |

---

## 五、列表字段定义

### 5.1 字段明细

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
|------|----------|----------|----------|--------|
| 1 | 操作者 | 文本 | 执行操作的用户名 | admin |
| 2 | 租户 | 文本 | 操作所属的租户，无则显示"-" | test01 |
| 3 | 操作行为 | 文本 | 具体的操作类型 | 登录 |
| 4 | 请求结果 | 标签 | 操作结果：已成功、已失败 | 已成功 |
| 5 | 操作时间 | 日期时间 | 操作发生的时间 | 2026-05-25 07:22:46 |
| 6 | 请求方法 | 文本 | HTTP请求方法 | POST |

### 5.2 操作行为类型

| 操作行为 | 说明 |
|----------|------|
| 登录 | 用户登录系统 |
| 创建任务 | 创建新的计算任务 |
| 删除任务 | 删除任务 |
| 资源分配 | 分配GPU资源给租户 |
| 资源释放 | 释放租户资源 |
| 节点上架 | 节点上架操作 |
| 节点下架 | 节点下架操作 |
| 配置变更 | 系统配置修改 |

### 5.3 请求结果说明

| 结果 | 说明 |
|------|------|
| 已成功 | 操作执行成功 |
| 已失败 | 操作执行失败 |

### 5.4 请求方法说明

| 方法 | 说明 |
|------|------|
| GET | 查询操作 |
| POST | 创建操作 |
| PUT | 更新操作 |
| DELETE | 删除操作 |

---

## 六、分页功能

### 6.1 分页控件

| 控件 | 说明 |
|------|------|
| 上一页 | 跳转到上一页，首页时禁用 |
| 页码列表 | 显示页码，当前页高亮，支持跳页 |
| 下一页 | 跳转到下一页，末页时禁用 |
| 每页条数 | 下拉选择：20条/页 |
| 前往 | 输入页码跳转 |

### 6.2 默认配置
- 默认每页：20条
- 默认当前页：第1页

---

## 七、业务规则

### 7.1 日志记录范围
- 记录所有用户的操作行为
- 包括成功和失败的请求
- 记录完整的请求信息

### 7.2 日志保留策略
- 日志默认保留一定周期
- 支持按时间范围查询历史日志

### 7.3 数据追溯
- 支持按操作者追溯用户行为
- 支持按租户追溯租户操作
- 支持按时间范围追溯问题

---

## 八、交互流程

```
进入审计日志页面
        │
        ↓
加载默认时间范围内的日志
        │
        ↓
显示日志列表
        │
        ├──→ 用户选择租户 → 过滤日志列表
        │
        ├──→ 用户选择用户 → 过滤日志列表
        │
        ├──→ 用户修改时间范围 → 重新查询
        │
        └──→ 用户点击日志下载 → 导出日志文件
```

---

## 九、安全要求

### 9.1 权限控制
- 仅管理员可查看审计日志
- 普通用户无权限访问

### 9.2 数据安全
- 日志数据只读，不可修改
- 导出文件需包含完整的审计信息
