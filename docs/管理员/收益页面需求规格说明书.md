# 收益（账单概览）页面需求规格说明书

## 一、页面概述

### 1.1 页面定位

收益页面是计费管理模块下的子页面，展示平台的账单统计信息，包括总金额、使用时长、支付情况等，支持按不同时间维度查看。

### 1.2 页面功能

- 展示账单概览统计（总金额、总使用时长、已支付）
- 支持按日/周/月/季度/年切换查看
- 展示费用趋势图表
- 展示账单明细列表
- 支持按租户筛选
- 支持导出账单数据

## 三、时间维度切换

### 3.1 维度选项

| 维度 | 说明               |
| ---- | ------------------ |
| 日   | 查看每日账单数据   |
| 周   | 查看每周账单汇总   |
| 月   | 查看每月账单汇总   |
| 季度 | 查看每季度账单汇总 |
| 年   | 查看年度账单汇总   |

### 3.2 切换逻辑

- 点击对应维度按钮切换
- 默认选中"日"
- 切换后自动刷新数据

## 八、业务规则

### 8.1 计费模式说明

- **按需计费**：按实际使用时长计费
- **包年/包月**：预付费固定周期计费

### 8.2 费用计算

- 费用 = 使用时长 × GPU单价 × GPU数量
- 包年费用按约定价格计算

## 九、交互流程

```
进入账单概览页面
        │
        ↓
加载默认时间维度（日）数据
        │
        ↓
显示统计指标和图表
        │
        ├──→ 用户切换时间维度 → 刷新数据
        │
        ├──→ 用户修改时间范围 → 查询历史数据
        │
        ├──→ 用户选择租户 → 过滤账单明细
        │
        ├──→ 用户点击导出 → 导出账单文件
        │
        └──→ 用户点击查看详情 → 进入账单详情页
```
