# 任务中心页面需求规格说明书（管理员版）

## 一、页面概述

### 1.1 页面定位
任务中心是管理员视角的核心业务模块，管理所有租户的计算任务，包括训练任务、模型微调、推理服务和开发机，支持跨租户查看和管理。

### 1.2 页面功能
- 展示所有租户的任务列表
- 支持按任务类型切换查看
- 支持按租户筛选
- 支持任务名称/ID搜索
- 支持批量删除任务
- 支持任务模板管理
- 支持任务回收站

## 四、最新任务（Tab1）

### 4.1 任务类型标签页

| 任务类型 | 说明 |
|----------|------|
| 训练任务 | 深度学习模型训练任务 |
| 模型微调 | 基于预训练模型的微调任务 |
| 推理服务 | 模型在线推理服务部署 |
| 开发机 | 交互式开发环境实例 |

### 4.2 筛选与搜索功能

| 筛选项 | 控件类型 | 说明 |
|--------|----------|------|
| 租户选择 | 下拉选择框 | 按租户筛选任务，可清空 |
| 任务名称/ID搜索 | 文本输入框 | 支持模糊搜索任务名称或ID |

### 4.3 批量操作

| 按钮 | 说明 |
|------|------|
| 批量删除 | 删除选中的多个任务，选中时可用 |
| 刷新 | 刷新当前任务列表 |

### 4.4 任务列表字段

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
|------|----------|----------|----------|--------|
| 1 | 选择框 | checkbox | 用于批量选择任务 | ✓ |
| 2 | 名称/ID | 文本 | 任务名称和系统生成的唯一ID | test / job-xxx |
| 3 | 任务类型 | 标签 | CPU、Simple、MPI等 | CPU |
| 4 | 状态 | 标签 | 运行中、停止、排队中、失败 | 运行中 |
| 5 | 就绪状态 | 文本 | 就绪实例数/总实例数 | 1/1 |
| 6 | 运行时长 | 文本 | 任务已运行时间 | 40h 1m 58s |
| 7 | 创建时间 | 日期时间 | 任务创建时间 | 2026-05-23 15:29:21 |
| 8 | 实例数量 | 数值 | 任务运行的实例数 | 1 |
| 9 | 任务优先级 | 标签 | 任务调度优先级 | high-priority-job |
| 10 | 回收策略 | 标签 | 可回收、不可回收 | 可回收 |
| 11 | GPU数量 | 数值 | 任务使用的GPU卡数 | 0 |
| 12 | 资源规格 | 文本 | GPU规格名称 | NVIDIA-GPU-HBM2E-80GB |
| 13 | 用户 | 文本 | 任务所属用户 | test-0415 |
| 14 | 租户/别名 | 文本 | 任务所属租户 | test-0415 |
| 15 | 操作 | 按钮 | 删除等操作 | [删除] |

### 4.5 任务状态说明

| 状态 | 颜色 | 说明 |
|------|------|------|
| 运行中 | 蓝色 | 任务正在执行中 |
| 停止 | 灰色 | 任务已停止执行 |
| 排队中 | 橙色 | 任务在队列中等待资源 |
| 失败 | 红色 | 任务执行失败 |

### 4.6 任务类型说明

| 类型 | 说明 |
|------|------|
| CPU | 仅使用CPU资源的任务 |
| Simple | 单实例任务 |
| MPI | MPI分布式任务 |

## 九、业务规则

### 9.1 任务创建规则
- 必须选择租户（租户有GPU配额）
- 必须配置资源规格
- 实例数量不能超过租户配额

### 9.2 任务删除规则
- 运行中的任务可停止后删除
- 删除后任务进入回收站
- 回收站中的任务可彻底删除

### 9.3 任务调度规则
- 高优先级任务优先调度
- 资源不足时任务进入排队状态
- 按任务创建顺序排队

## 十、交互流程

### 10.1 查看任务列表

```
进入任务中心页面
        │
        ↓
加载最新任务标签页
        │
        ↓
显示训练任务列表（默认）
        │
        ├──→ 用户切换任务类型 → 显示对应类型任务
        │
        ├──→ 用户选择租户筛选 → 过滤任务列表
        │
        ├──→ 用户搜索任务名称 → 过滤任务列表
        │
        └──→ 用户切换标签页 → 显示模板或回收站
```

### 10.2 删除任务流程

```
选中要删除的任务
        │
        ↓
点击"删除"或"批量删除"
        │
        ↓
弹出确认对话框
        │
   ┌────┴────┐
   ↓         ↓
  取消      确认
   │         │
   ↓         ↓
关闭对话框 停止任务（如运行中）→ 移入回收站 → 刷新列表
```

### 10.3 任务模板流程

```
切换到任务模板标签页
        │
        ↓
显示模板列表
        │
        ├──→ 点击"创建任务" → 基于模板创建新任务
        │
        ├──→ 点击"修改" → 编辑模板配置
        │
        └──→ 点击"删除" → 删除模板
```

## 十一、管理员特有功能

### 11.1 跨租户管理
- 管理员可查看所有租户的任务
- 支持按租户筛选
- 可对任意租户的任务进行删除操作

### 11.2 权限控制
- 只有管理员可以访问任务中心管理页面
- 操作需要二次确认
- 高危操作有风险提示
