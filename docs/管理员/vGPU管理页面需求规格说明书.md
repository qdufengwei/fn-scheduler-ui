# vGPU管理页面需求规格说明书

## 一、页面概述

### 1.1 页面定位

vGPU管理页面是集群资源模块下的子页面，用于配置和管理虚拟GPU（vGPU）规格。vGPU是通过软件虚拟化技术将物理GPU资源切分成多个逻辑GPU实例，每个实例具有独立的算力和显存配置，适用于需要细粒度GPU资源分配的场景。

### 1.2 页面功能

- 展示所有已配置的vGPU规格列表
- 支持新增vGPU规格配置
- 支持搜索已有规格
- 展示vGPU规格的使用情况统计

## 三、筛选与搜索功能

### 3.1 搜索条件

| 搜索项 | 控件类型 | 说明 |
| --- | --- | --- |
| 搜索vGPU规格名称 | 文本输入框 | 支持模糊搜索vGPU规格的名称，如输入"A10"可搜索包含"A10"的规格 |

### 3.2 搜索逻辑

- 输入关键字后实时过滤列表，或按回车键触发搜索
- 搜索范围为"规格名称"字段
- 支持部分匹配，不区分大小写

## 四、列表字段定义

### 4.1 字段明细

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
| --- | --- | --- | --- | --- |
| 1 | 规格名称 | 文本 | vGPU规格的唯一标识名称，格式通常为：`{GPU型号}-{算力百分比}-{显存大小}` | A10-25%-8G |
| 2 | 资源规格 | 文本 | 该vGPU规格所基于的物理GPU型号 | NVIDIA-GPU-HBM2E-80GB |
| 3 | 状态 | 标签 | vGPU规格的启用状态，枚举值：启用、禁用 | 启用 |
| 4 | 算力 | 数值+单位 | 每个vGPU实例分配的算力百分比 | 25% |
| 5 | 显存 | 数值+单位 | 每个vGPU实例分配的显存大小 | 8 GiB |
| 6 | 用户购买 | 数值 | 用户通过购买方式获得的该规格vGPU实例数量 | 2 |
| 7 | 分配 | 数值 | 管理员分配（白名单）的该规格vGPU实例数量 | 5 |
| 8 | 使用中 | 数值 | 当前正在使用的vGPU实例数量 | 3 |
| 9 | 实例总数 | 数值 | 该规格vGPU实例的总数量（用户购买+分配） | 7 |
| 10 | 单价 | 货币 | 该规格vGPU的计费单价 | ¥1.00 / 时 |
| 11 | 操作 | 按钮 | 可执行的操作按钮 | 编辑、删除 |

### 4.2 字段业务含义

#### 算力字段说明

- 表示每个vGPU实例占物理GPU算力的百分比
- 例如：25%表示该vGPU实例可使用物理GPU 25%的计算能力
- 取值范围：1% - 100%

#### 显存字段说明

- 表示每个vGPU实例分配的显存大小
- 单位：GiB（Gibibyte，二进制单位）
- 不能超过物理GPU的总显存

#### 用户购买 vs 分配

- **用户购买**：租户通过付费购买获得的vGPU实例数量
- **分配**：管理员通过白名单机制分配给租户的vGPU实例数量
- **实例总数** = 用户购买 + 分配

## 五、新增vGPU规格弹窗

### 5.1 入口触发

点击"添加vGPU规格"按钮，进入新增vGPU规格页面。

### 5.2 安全提示对话框

新增vGPU规格前，系统会弹出安全提示对话框：

```
┌─────────────────────────────────────────────────────────────────────┐
│  提示                                                          [X] │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  vGPU存在隔离性和安全性风险，可能导致资源边界模糊及数据安全风险，    │
│  请谨慎使用。                                                       │
│                                                                     │
│                    ┌────────┐    ┌────────┐                         │
│                    │  取消  │    │  确认  │                         │
│                    └────────┘    └────────┘                         │
└─────────────────────────────────────────────────────────────────────┘
```

- 点击"取消"：关闭对话框，不进入新增页面
- 点击"确认"：关闭对话框，显示新增vGPU规格表单

### 5.3 新增表单字段

| 序号 | 字段名称 | 是否必填 | 控件类型 | 说明 |
| --- | --- | --- | --- | --- |
| 1 | 资源规格 | 是 | 下拉选择框 | 选择物理GPU型号，如NVIDIA-GPU-HBM2E-80GB |
| 2 | GPU节点 | 是 | 下拉选择框 | 选择要创建vGPU的物理节点，仅显示支持vGPU的节点 |
| 3 | 每个vGPU的算力 | 是 | 滑动条 | 拖动滑块选择算力百分比，范围1%-100% |
| 4 | 每个vGPU的显存 | 是 | 数字输入框（带加减按钮） | 设置显存大小，单位GiB |
| 5 | 规格名称 | 是 | 文本框（自动生成） | 根据上述配置自动生成，格式：GPU-{算力百分比}-{显存大小}GiB |

### 5.4 字段交互逻辑

#### 资源规格

- 下拉列表展示系统中所有可用的物理GPU型号
- 选择后，"GPU节点"下拉框会过滤出支持该GPU型号的节点

#### GPU节点

- 下拉列表展示支持vGPU功能的物理节点
- 选择后，系统会校验该节点是否满足vGPU创建条件

#### 每个vGPU的算力

- 使用滑动条控件，默认值为1%
- 拖动时实时显示当前百分比数值
- 范围：1% - 100%

#### 每个vGPU的显存

- 数字输入框，带增加/减少按钮
- 默认值：1 GiB
- 单位：GiB
- 点击"+"按钮增加显存值
- 点击"-"按钮减少显存值
- 最大值不能超过物理GPU总显存

#### 规格名称

- 自动生成，不可手动编辑
- 生成规则：`GPU-{算力百分比}-{显存值}GiB`
- 示例：算力设为1%，显存设为1GiB，则规格名称为：`GPU-1%-1GiB`

### 5.5 表单操作按钮

| 按钮               | 功能                       |
| ------------------ | -------------------------- |
| 确认               | 提交表单，创建vGPU规格     |
| 取消               | 取消操作，返回vGPU列表页面 |
| 返回箭头（左上角） | 返回上一页（同取消功能）   |

## 六、编辑vGPU规格弹窗

### 6.1 入口触发

在vGPU列表中，点击某一行的"编辑"操作按钮，进入编辑页面。

### 6.2 编辑表单字段

编辑表单与新增表单字段相同，区别在于：

- 所有字段回显当前vGPU规格的配置值
- 可修改的字段与新增时一致
- 规格名称仍为自动生成，根据修改后的配置实时更新

## 九、业务规则

### 9.1 vGPU规格命名规则

- 自动生成，格式：`GPU-{算力百分比}-{显存值}GiB`
- 示例：
  - `GPU-25%-8GiB`：表示算力25%，显存8GiB的规格
  - `GPU-50%-16GiB`：表示算力50%，显存16GiB的规格

### 9.2 vGPU实例数量统计规则

- **实例总数** = 用户购买 + 分配
- **使用中** ≤ 实例总数
- **剩余可用** = 实例总数 - 使用中

### 9.3 vGPU与物理GPU关系

- 一张物理GPU可切分为多个vGPU实例
- 所有vGPU实例的算力之和不应超过100%
- 所有vGPU实例的显存之和不应超过物理GPU总显存

### 9.4 删除规则

- 有正在使用的vGPU实例时，不可删除该规格
- 删除前需确认该规格下无使用中的实例

## 十、页面交互流程

### 10.1 新增vGPU规格流程

```
点击"添加vGPU规格"
        │
        ↓
弹出安全提示对话框
        │
   ┌────┴────┐
   ↓         ↓
 点击取消   点击确认
   │         │
   ↓         ↓
 返回列表   显示新增表单
              │
              ↓
          填写表单字段
              │
              ↓
          点击"确认"
              │
              ↓
          提交创建请求
              │
         ┌────┴────┐
         ↓         ↓
      创建成功   创建失败
         │         │
         ↓         ↓
      返回列表   显示错误提示
```

### 10.2 编辑vGPU规格流程

```
点击"编辑"按钮
        │
        ↓
显示编辑表单（回显当前配置）
        │
        ↓
修改配置字段
        │
        ↓
点击"确认"提交
        │
        ↓
更新成功，返回列表
```

## 十一、UI设计规范

### 11.1 页面标题

- 位置：页面左上角
- 内容：vGPU
- 样式：标准页面标题样式

### 11.2 操作区域

- 位置：标题下方
- 布局：搜索框左侧，添加按钮右侧

### 11.3 表格样式

- 表头：固定表头，滚动时保持可见
- 行高：标准行高，便于阅读
- 斑马纹：建议使用，增强可读性
- 操作列：固定在表格右侧

### 11.4 表单样式

- 标签位置：字段左侧，必填项标注红色星号
- 输入控件：统一宽度，保持对齐
- 错误提示：字段下方显示红色错误提示文本
