# 服务配置页面需求规格说明书

## 一、页面概述

### 1.1 页面定位

服务配置页面是集群资源模块下的子页面，用于设置平台运行参数，包括应用配置、任务日志配置、默认镜像配置等系统级参数。

### 1.2 页面功能

- 配置应用运行参数（内存上限、CPU上限）
- 配置任务日志参数
- 配置默认镜像地址
- 配置CPU节点标签
- 配置镜像拉取认证

---

## 三、应用配置

### 3.1 内存上限

| 属性     | 说明                                 |
| -------- | ------------------------------------ |
| 配置项   | 内存上限                             |
| 控件类型 | 数字输入框                           |
| 默认值   | 150                                  |
| 单位     | GiB                                  |
| 说明     | 用于定义任务中单个实例使用的内存上限 |
| 必填     | 是                                   |

### 3.2 CPU上限

| 属性     | 说明                                |
| -------- | ----------------------------------- |
| 配置项   | CPU上限                             |
| 控件类型 | 数字输入框                          |
| 默认值   | 32                                  |
| 单位     | 核                                  |
| 说明     | 用于定义任务中单个实例使用的CPU上限 |
| 必填     | 是                                  |

### 3.3 CPU节点标签

| 属性     | 说明                     |
| -------- | ------------------------ |
| 配置项   | CPU节点标签              |
| 控件类型 | 键值对输入（可添加多条） |
| 默认值   | cpu_node: true           |
| 说明     | 用于标识CPU节点          |
| 操作     | 支持添加、删除标签       |

### 3.4 算法开发访问IP

| 属性     | 说明                         |
| -------- | ---------------------------- |
| 配置项   | 算法开发访问IP               |
| 控件类型 | 文本输入框                   |
| 默认值   | 10.1.2.108                   |
| 说明     | 用于指定访问算法服务的IP地址 |
| 必填     | 是                           |

### 3.5 使用认证拉取任务镜像

| 属性     | 说明                         |
| -------- | ---------------------------- |
| 配置项   | 使用认证拉取任务镜像         |
| 控件类型 | 开关                         |
| 默认值   | 不使用认证                   |
| 说明     | 开启后任务拉取镜像时需要认证 |

---

## 七、业务规则

### 7.1 配置生效规则

- 配置修改后立即生效
- 对新创建的任务应用新配置
- 已运行的任务不受影响

### 7.2 参数校验规则

- 内存上限：必须为正整数
- CPU上限：必须为正整数
- IP地址：必须为有效的IP格式
- 镜像地址：必须为有效的镜像URL格式

### 7.3 镜像配置说明

- 默认镜像用于创建开发机和云终端
- 用户可自行指定镜像覆盖默认值

## 八、交互流程

```
进入服务配置页面
        │
        ↓
加载当前配置值
        │
        ↓
显示所有配置项
        │
        ├──→ 用户修改配置值
        │
        ├──→ 用户添加/删除CPU节点标签
        │
        └──→ 用户点击确认
                │
                ↓
            校验输入值
                │
           ┌────┴────┐
           ↓         ↓
          失败      成功
           │         │
           ↓         ↓
        显示错误   保存配置 → 提示成功
```
