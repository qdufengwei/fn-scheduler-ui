# 镜像中心页面需求规格说明书（租户版）

## 一、页面概述

### 1.1 页面定位
镜像中心页面管理租户可用的容器镜像，包括公共镜像和私有镜像，为任务运行提供环境配置。

### 1.2 页面功能
- 展示公共镜像列表
- 展示私有镜像列表
- 创建自定义镜像
- 删除镜像

## 八、业务规则

### 8.1 镜像使用规则
- 公共镜像所有租户可使用
- 私有镜像仅创建租户可用
- 镜像用于定义任务运行环境

### 8.2 镜像创建规则
- 需指定镜像仓库地址
- 支持从开发机保存镜像

---

## 九、交互流程

### 9.1 查看镜像列表

```
进入镜像中心页面
        │
        ↓
加载镜像仓库标签页
        │
        ↓
显示公共镜像列表（默认）
        │
        ├──→ 用户切换到私有镜像 → 显示私有镜像
        │
        └──→ 用户点击镜像名称 → 查看镜像详情和版本
```

### 9.2 创建镜像流程

```
点击"创建镜像"
        │
        ↓
弹出创建表单
        │
        ↓
填写镜像名称和仓库信息
        │
        ↓
点击"确认" → 创建镜像 → 刷新列表
```
