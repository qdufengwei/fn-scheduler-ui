# 我的资源页面需求规格说明书（租户版）

## 一、页面概述

### 1.1 页面定位

我的资源页面展示租户购买的GPU资源实例信息，包括资源规格、分配方式、使用状态和计费信息，是租户管理自身计算资源的主要入口。

### 1.2 页面功能

- 展示租户拥有的GPU资源实例
- 查看资源详情
- 配置MIG（多实例GPU）
- 关闭MIG功能

## 三、资源实例列表字段

### 3.1 字段明细

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
| --- | --- | --- | --- | --- |
| 1 | 资源规格 | 文本 | GPU型号规格名称 | NVIDIA-GPU-HBM2E-80GB |
| 2 | 资源状态 | 标签 | 资源分配方式：整机分配、弹性分配 | 弹性分配 |
| 3 | 资源用量 | 文本 | 已使用/总量GPU卡数 | 1 卡 |
| 4 | GPU数量 | 数值 | 当前实例拥有的GPU卡数 | 1 |
| 5 | 购买周期 | 标签 | 购买方式：按需(小时)、包月、包年 | 按需(小时) |

| 周期       | 说明                   |

## 七、业务规则

### 7.1 资源展示规则

- 仅展示当前租户购买的资源
- 按创建时间降序排列
- 实时更新资源用量状态

### 7.2 MIG配置规则

- 仅支持支持MIG的GPU型号（如A100、H100）
- MIG分割后每个实例独立使用
- MIG实例数量受GPU型号限制

### 7.3 计费规则

- 按需计费资源按时长计费
- 包年包月资源到期后自动释放
- 资源单价在购买时确定

## 八、交互流程

### 8.1 查看资源列表

```
进入我的资源页面
        │
        ↓
加载租户的资源实例列表
        │
        ↓
显示资源规格、状态、用量等信息
```

### 8.2 MIG配置流程

```
点击"MIG配置"按钮
        │
        ↓
弹出MIG配置弹窗
        │
        ↓
选择MIG规格和实例数量
        │
        ↓
点击"确认"
        │
        ↓
配置MIG → 更新资源列表
```

### 8.3 查看详情流程

```
点击"查看详情"按钮
        │
        ↓
弹出资源详情弹窗
        │
        ↓
展示资源详细信息
        │
        │   包括：GPU型号、使用状态、绑定任务等
        │
        ↓
点击"关闭" → 关闭弹窗
```

## 参考平台对齐修订（2026-05-26，原型版）

- 对齐基准：`https://aicloud.infrawaves.com/` 普通用户账号（`test01`）当前页面结构与交互。
- 原型策略：优先对齐页面信息架构、核心控件、关键按钮与状态展示；暂不接入真实后端数据。
- 交互规范：
  - 列表页统一包含筛选区、操作区、数据表格、分页。
  - 创建/编辑类操作使用右侧抽屉（Drawer）承载表单。
  - 确认类操作使用气泡确认（Popconfirm）。
- 数据口径：统计卡、趋势图、监控值、费用值当前均允许为模拟数据或空状态展示，用于产品原型演示。
- 对齐边界：若线上页面存在高耦合业务校验（鉴权、计费结算、实时监控流），原型仅保留界面入口与交互壳层。
