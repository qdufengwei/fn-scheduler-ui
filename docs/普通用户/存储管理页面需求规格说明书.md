# 存储管理页面需求规格说明书（租户版）

## 一、页面概述

### 1.1 页面定位
存储管理页面用于管理租户的持久化存储卷（PVC），为任务提供数据持久化能力。

### 1.2 页面功能
- 展示存储卷列表
- 创建新的存储卷
- 更新存储卷容量
- 删除存储卷

## 四、存储卷列表字段

### 4.1 字段明细

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
|------|----------|----------|----------|--------|
| 1 | 名称 | 文本 | 存储卷名称 | test |
| 2 | 类型 | 文本 | 存储类型 | example.com/nfs |
| 3 | 已使用的量 | 文本 | 已使用容量 | 5 GB |
| 4 | 容量 | 文本 | 总容量 | 5 GB |
| 5 | 状态 | 标签 | 存储卷状态 | Bound |
| 6 | 创建时间 | 日期时间 | 存储卷创建时间 | 2026-03-03 02:24:23 |

### 4.2 存储状态说明

| 状态 | 说明 |
|------|------|
| Bound | 存储卷已绑定，可使用 |
| Available | 存储卷可用，未绑定 |
| Pending | 存储卷等待中 |

## 八、业务规则

### 8.1 存储卷创建规则
- 存储卷名称唯一
- 容量不能超过配额限制

### 8.2 存储卷使用规则
- 存储卷可挂载到多个任务
- 支持动态扩容（取决于存储类型）

### 8.3 删除规则
- 已挂载的存储卷需先卸载
- 删除后数据不可恢复

## 九、交互流程

```

        │
        ↓

        │
        ↓

        │
        ↓
点击"确认" → 更新存储卷 → 刷新列表
```
