# 开发机页面需求规格说明书（租户版）

## 一、页面概述

### 1.1 页面定位

开发机页面用于管理交互式开发环境实例，提供Jupyter Notebook、VSCode等开发工具，是租户进行模型开发和调试的主要入口。

### 1.2 页面功能

- 展示开发机列表
- 创建开发机实例
- 保存镜像
- 删除开发机
- 连接开发环境

## 八、业务规则

### 8.1 开发机创建规则

- 必须指定开发机类型（Jupyter/VSCode）
- 必须选择镜像和资源配置
- 名称在同一租户下唯一

### 8.2 资源分配规则

- 资源不足时开发机进入排队状态
- 排队中的开发机不占用资源

### 8.3 镜像保存规则

- 仅运行中的开发机可保存镜像
- 保存镜像需要一定时间

## 九、交互流程

### 9.1 创建开发机

```
点击"创建开发机"
        │
        ↓
弹出创建表单
        │
        ↓
填写名称、选择类型
        │
        ↓
选择镜像和资源配置
        │
        ↓
点击"确认"
        │
        ↓
创建开发机 → 刷新列表
        │
        ├──→ 资源充足 → 状态变为"运行中"
        │
        └──→ 资源不足 → 状态变为"排队中"
```

### 9.2 连接开发机

```
开发机状态为"运行中"
        │
        ↓
点击开发机名称或连接按钮
        │
        ↓
打开Jupyter/VSCode界面
```

### 9.3 保存镜像流程

```
点击"保存镜像"
        │
        ↓
弹出保存镜像表单
        │
        ↓
填写镜像名称和描述
        │
        ↓
点击"确认"
        │
        ↓
开始保存镜像 → 显示进度 → 完成后提示
```

### 9.4 删除开发机

```
点击"删除"
        │
        ↓
弹出确认对话框
        │
   ┌────┴────┐
   ↓         ↓
  取消      确认
   │         │
   ↓         ↓
关闭对话框 停止开发机 → 删除 → 刷新列表
```
