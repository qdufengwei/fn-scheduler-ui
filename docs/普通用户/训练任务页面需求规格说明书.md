# 训练任务页面需求规格说明书（普通用户版）

## 一、页面概述

### 1.1 页面定位

训练任务页面是普通用户创建和管理深度学习模型训练任务的主要入口，支持多种分布式训练框架，提供任务的完整生命周期管理。

### 1.2 页面功能

- 展示当前用户及同租户的训练任务列表
- 创建新的训练任务
- 批量删除任务
- 查看任务详情
- 任务模板管理
- 任务回收站管理

## 四、最新任务（Tab1）

### 4.1 筛选与搜索功能

| 筛选项          | 控件类型   | 说明                     |
| --------------- | ---------- | ------------------------ |
| 全部/我创建的   | 单选按钮组 | 筛选任务归属，默认"全部" |

| 按钮     | 说明                           |
| -------- | ------------------------------ |
| 创建任务 | 创建新的训练任务，弹出右侧抽屉 |
| 批量删除 | 删除选中的多个任务，选中时可用 |
| 刷新     | 刷新任务列表                   |

### 4.3 任务列表字段

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
| --- | --- | --- | --- | --- |
| 1 | 选择框 | checkbox | 用于批量选择任务 | ✓ |
| 2 | 名称/ID | 文本 | 任务名称和唯一ID | my-training / job-xxx |
| 3 | 任务类型 | 标签 | MPI、PyTorch、Simple、CPU、Deployment | MPI |
| 4 | 状态 | 标签 | 运行中、停止、排队中、失败 | 运行中 |
| 5 | 就绪状态 | 文本 | 就绪实例数/总实例数 | 1/1 |
| 6 | 运行时长 | 文本 | 任务已运行时间 | 40h 1m 58s |
| 7 | 创建时间 | 日期时间 | 任务创建时间 | 2026-05-23 15:29:21 |
| 8 | 实例数量 | 数值 | 任务运行的实例数 | 2 |
| 9 | 任务优先级 | 数值 | 任务调度优先级(1-6) | 6 |
| 10 | 回收策略 | 标签 | 可回收、不可回收 | 不可回收 |
| 11 | GPU数量 | 数值 | 任务使用的GPU卡数 | 2 |
| 12 | 资源规格 | 文本 | GPU规格名称 | NVIDIA-GPU-HBM2E-80GB |
| 13 | 用户 | 文本 | 任务创建者 | test01 |

### 4.4 任务类型说明

| 类型       | 说明              |
| ---------- | ----------------- |
| MPI        | MPI分布式训练任务 |
| PyTorch    | PyTorch分布式训练 |
| Simple     | 单实例任务        |
| CPU        | 仅使用CPU的任务   |
| Deployment | 部署类型任务      |

### 4.5 任务状态说明

| 状态   | 颜色 | 说明                 |
| ------ | ---- | -------------------- |
| 运行中 | 蓝色 | 任务正在执行中       |

## 七、创建任务页面

### 7.1 页面布局

创建任务通过右侧抽屉（Drawer）展示，包含以下配置区块：

| 区块     | 说明                                       |
| -------- | ------------------------------------------ |
| 基础信息 | 任务名称、描述、容忍度                     |
| 资源配置 | 实例配置、优先级、回收策略、自动重试、存储 |
| 环境配置 | 镜像、工作目录、命令、环境变量             |

### 7.2 基础信息

| 字段名称 | 是否必填 | 控件类型 | 说明 |
| --- | --- | --- | --- |
| 任务名称 | 是 | 文本输入框 | 仅支持中文、字母、数字及 -、\_、/、() 符号 |
| 描述 | 否 | 文本域 | 任务描述，最多1024字符 |
| 容忍度 | 否 | 配置项 | 设置节点容忍度 |

### 7.3 实例配置

#### 7.3.1 任务类型选择

| 类型       | 说明                                  |
| ---------- | ------------------------------------- |
| MPI        | MPI分布式训练，包含master和worker角色 |
| PyTorch    | PyTorch分布式训练                     |
| Simple     | 单实例任务                            |
| CPU        | 仅CPU任务                             |
| Deployment | 部署任务                              |

#### 7.3.2 GPU模式选项

| 选项        | 说明          |
| ----------- | ------------- |
| 启用MIG模式 | 使用多实例GPU |
| 整机调度    | 整卡调度模式  |
| vGPU        | 虚拟GPU模式   |

#### 7.3.3 实例配置字段

| 字段      | 说明                      |
| --------- | ------------------------- |
| Roles     | 实例角色（master/worker） |
| 实例数量  | 该角色的实例数            |
| 资源规格  | GPU型号规格               |
| GPU数量   | 每实例GPU卡数             |
| CPU(核)   | 每实例CPU核数             |
| 内存(GiB) | 每实例内存大小            |

### 7.4 优先级配置

| 配置项 | 说明                    |
| ------ | ----------------------- |
| 优先级 | 1-6，数值越大优先级越高 |

### 7.5 回收策略

| 配置项   | 说明                   |
| -------- | ---------------------- |
| 回收策略 | 开启后任务可被系统回收 |
| 自动重试 | 任务失败后自动重试     |

### 7.6 存储配置

| 存储类型 | 说明             |
| -------- | ---------------- |
| 存储卷   | 持久化存储卷     |
| 临时卷   | EmptyDir临时存储 |
| S3       | S3对象存储       |

### 7.7 环境配置

| 字段名称 | 是否必填 | 控件类型        | 说明                 |
| -------- | -------- | --------------- | -------------------- |
| 镜像     | 是       | 下拉选择/输入框 | 预置镜像或自定义镜像 |
| 工作目录 | 是       | 文本输入框      | 命令执行目录         |
| 命令模式 | 是       | 单选            | 自定义模式/shell模式 |
| 入口命令 | 是       | 文本输入        | 执行的命令           |
| 环境变量 | 否       | 键值对配置      | 环境变量设置         |

### 7.8 操作按钮

| 按钮     | 功能               |
| -------- | ------------------ |
| 确认     | 创建任务           |
| 存为模版 | 保存当前配置为模板 |

## 九、业务规则

### 9.1 任务创建规则

- 必须选择资源规格（需先购买资源）
- 任务名称在同一租户下唯一
- 优先级影响任务调度顺序

### 9.2 任务删除规则

- 运行中的任务需先停止再删除
- 删除后任务进入回收站
- 回收站中的任务可彻底删除

### 9.3 资源检查规则

- 创建任务前检查资源配额
- 资源不足时提示购买
- 实时显示资源使用情况

## 十、交互流程

### 10.1 创建训练任务

```
点击"创建任务"
        │
        ↓
弹出右侧抽屉
        │
        ↓
填写基础信息（名称、描述）
        │
        ↓
配置资源（任务类型、实例、GPU）
        │
        ↓
配置存储（存储卷、临时卷、S3）
        │
        ↓
配置环境（镜像、命令、环境变量）
        │
        ↓
点击"确认"
        │
        ↓
创建任务 → 关闭抽屉 → 刷新列表
```

### 10.2 删除任务流程

```
选中要删除的任务
        │
        ↓
点击"删除"或"批量删除"
        │
        ↓
弹出确认对话框（Popconfirm）
        │
   ┌────┴────┐
   ↓         ↓
  取消      确认
   │         │
   ↓         ↓
关闭对话框 停止任务 → 移入回收站 → 刷新列表
```

### 10.3 使用任务模板

```
切换到任务模板标签页
        │
        ↓
显示模板列表
        │
        ├──→ 点击"创建任务" → 弹出抽屉，自动填充模板配置
        │
        ├──→ 点击"修改" → 弹出抽屉，编辑模板配置
        │
        └──→ 点击"删除" → 二次确认 → 删除模板
```

## 十一、交互规范

### 11.1 确认类交互

- 删除操作使用气泡确认（Popconfirm）
- 批量删除使用模态确认框

### 11.2 表单类交互

- 创建/编辑任务使用右侧抽屉（Drawer）
- 抽屉宽度固定，支持滚动

### 11.3 列表交互

- 列表支持多选（checkbox）
- 表格支持横向滚动
- 分页控件固定在底部

## 十二、参考平台对齐修订（2026-05-26）

### 12.1 对齐基准

- 对齐平台：`https://aicloud.infrawaves.com/`
- 对齐账号：普通用户账号（`test01`）
- 原型策略：优先对齐页面信息架构、核心控件、关键按钮与状态展示

### 12.2 交互规范

- 列表页统一包含筛选区、操作区、数据表格、分页

### 12.3 数据口径

- 统计卡、趋势图、监控值、费用值当前均允许为模拟数据或空状态展示

- 若线上页面存在高耦合业务校验（鉴权、计费结算、实时监控流）
- 原型仅保留界面入口与交互壳层

### 12.5 页面级对齐要点

- 线上URL：`/task-training/list`
