# 模型服务市场页面需求规格说明书（租户版）

## 一、页面概述

### 1.1 页面定位

模型服务市场页面提供模型即服务（MaaS）能力，租户可以浏览和使用其他用户发布的模型推理服务，通过API调用模型能力。

### 1.2 页面功能

- 浏览可用模型服务
- 按分类筛选模型服务
- 查看模型服务详情和价格
- 调用模型推理API

## 三、筛选功能

### 3.1 分类筛选

| 分类     | 说明               |
| -------- | ------------------ |
| 文本生成 | 文本生成类模型服务 |
| 代码生成 | 代码生成类模型服务 |
| 视觉识别 | 图像识别类模型服务 |
| 语音处理 | 语音处理类模型服务 |

### 3.2 排序字段

| 排序字段 | 说明                |
| -------- | ------------------- |
| 输入价格 | 按输入token价格排序 |
| 输出价格 | 按输出token价格排序 |
| 创建时间 | 按创建时间排序      |
| 调用次数 | 按调用次数排序      |

### 3.3 排序方向

| 方向 | 说明         |
| ---- | ------------ |
| 升序 | 从低到高排序 |
| 降序 | 从高到低排序 |

## 六、分页功能

### 6.1 分页控件

| 控件     | 说明                     |
| -------- | ------------------------ |
| 上一页   | 跳转到上一页，首页时禁用 |

## 七、业务规则

### 7.1 服务浏览规则

- 显示租户可访问的模型服务
- 服务按筛选条件排序展示

### 7.2 服务调用规则

- 调用服务需有可用API KEY
- 按调用token数量计费

---

## 八、交互流程

### 8.1 浏览模型服务

```
进入模型服务市场
        │
        ↓
设置筛选条件（分类、排序）
        │
        ↓
显示符合条件的模型服务列表
        │
        ├──→ 点击服务 → 查看详情和调用示例
        │
        └──→ 调整筛选条件 → 更新列表
```

### 8.2 调用模型服务

```
选择要调用的模型服务
        │
        ↓
查看服务详情和API示例
        │
        ↓
使用API KEY发起调用
        │
        ↓
返回推理结果 → 记录调用次数和费用
```

## 参考平台对齐修订（2026-05-26，原型版）

- 对齐基准：`https://aicloud.infrawaves.com/` 普通用户账号（`test01`）当前页面结构与交互。
- 原型策略：优先对齐页面信息架构、核心控件、关键按钮与状态展示；暂不接入真实后端数据。
- 交互规范：
  - 列表页统一包含筛选区、操作区、数据表格、分页。
  - 创建/编辑类操作使用右侧抽屉（Drawer）承载表单。
  - 确认类操作使用气泡确认（Popconfirm）。
- 数据口径：统计卡、趋势图、监控值、费用值当前均允许为模拟数据或空状态展示，用于产品原型演示。
- 对齐边界：若线上页面存在高耦合业务校验（鉴权、计费结算、实时监控流），原型仅保留界面入口与交互壳层。
