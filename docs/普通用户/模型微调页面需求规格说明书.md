# 模型微调页面需求规格说明书（租户版）

## 一、页面概述

### 1.1 页面定位
模型微调页面用于管理基于预训练模型的微调任务，支持用户在已有模型基础上进行针对性训练，适应特定应用场景。

### 1.2 页面功能
- 展示模型微调任务列表
- 创建微调任务
- 批量删除任务
- 查看任务状态和进度

---

## 三、筛选与搜索功能

### 3.1 筛选项

| 筛选项 | 控件类型 | 说明 |
|--------|----------|------|
| 全部/我创建的 | 单选按钮 | 筛选任务归属 |
| 任务名称/ID搜索 | 文本输入框 | 支持模糊搜索 |

### 3.2 功能按钮

| 按钮 | 说明 |
|------|------|
| 创建微调任务 | 创建新的微调任务 |
| 批量删除 | 删除选中的多个任务 |

## 四、微调任务列表字段

### 4.1 字段明细

| 序号 | 字段名称 | 字段类型 | 字段说明 | 示例值 |
|------|----------|----------|----------|--------|
| 1 | 选择框 | checkbox | 用于批量选择任务 | ✓ |
| 2 | 名称/ID | 文本 | 任务名称和唯一ID | my-finetune / job-xxx |
| 3 | 状态 | 标签 | 运行中、停止、排队中、失败 | 运行中 |
| 4 | 模型名称 | 文本 | 基础模型名称 | Qwen3-4B |
| 5 | 就绪状态 | 文本 | 就绪实例数/总实例数 | 1/1 |
| 6 | 运行时长 | 文本 | 任务已运行时间 | 2h 30m |
| 7 | 创建时间 | 日期时间 | 任务创建时间 | 2026-05-23 15:29:21 |
| 8 | 实例数量 | 数值 | 任务运行的实例数 | 1 |
| 9 | GPU数量 | 数值 | 任务使用的GPU卡数 | 1 |
| 10 | 资源规格 | 文本 | GPU规格名称 | NVIDIA-GPU-HBM2E-80GB |
| 11 | 用户 | 文本 | 任务创建者 | test01 |
| 12 | 操作 | 按钮组 | 查看、删除等 | [查看] [删除] |

### 4.2 任务状态说明

| 状态 | 说明 |
|------|------|
| 运行中 | 微调任务正在执行 |
| 停止 | 任务已停止 |
| 排队中 | 等待资源分配 |
| 失败 | 任务执行失败 |

## 五、创建微调任务

### 5.1 表单字段

| 字段名称 | 是否必填 | 说明 |
|----------|----------|------|
| 任务名称 | 是 | 微调任务名称 |
| 基础模型 | 是 | 选择预训练模型 |
| 微调方法 | 是 | 选择微调方法（LoRA、全量等） |
| 数据集 | 是 | 训练数据集 |
| 资源配置 | 是 | GPU规格和数量 |
| 超参数配置 | 否 | 训练超参数设置 |

## 六、分页功能

### 6.1 分页控件

| 控件 | 说明 |
|------|------|
| 上一页 | 跳转到上一页，首页时禁用 |
| 页码列表 | 显示页码，当前页高亮 |
| 下一页 | 跳转到下一页，末页时禁用 |
| 每页条数 | 下拉选择：10条/页 |

---

## 七、业务规则

### 7.1 微调任务规则
- 必须选择基础模型
- 必须指定训练数据集
- 资源规格需匹配模型需求

### 7.2 模型选择规则
- 支持选择预置模型或自定义模型
- 模型需支持所选微调方法

---

## 八、交互流程

### 8.1 创建微调任务

```
点击"创建微调任务"
        │
        ↓

        │

        │
```

## 参考平台对齐修订（2026-05-26，原型版）

- 对齐基准：`https://aicloud.infrawaves.com/` 普通用户账号（`test01`）当前页面结构与交互。
- 原型策略：优先对齐页面信息架构、核心控件、关键按钮与状态展示；暂不接入真实后端数据。
- 交互规范：
  - 列表页统一包含筛选区、操作区、数据表格、分页。
  - 创建/编辑类操作使用右侧抽屉（Drawer）承载表单。
  - 确认类操作使用气泡确认（Popconfirm）。
- 数据口径：统计卡、趋势图、监控值、费用值当前均允许为模拟数据或空状态展示，用于产品原型演示。
- 对齐边界：若线上页面存在高耦合业务校验（鉴权、计费结算、实时监控流），原型仅保留界面入口与交互壳层。
