## 3. 逐页差异检查（原型视角）

### A. 我的资源
- 线上关键点：
  - 首页欢迎语
  - 任务统计卡（训练/开发机/推理/微调）
  - 计费卡（总使用量/待支付/已支付 + 周期）
  - 消息列表
  - 资源概览（标准/MIG/vGPU）
- 当前原型状态：
  - 已有资源表格与资源概览卡
  - 缺少消息列表、计费模块细节
- 差异优先级：P1

### B. 训练任务
- 线上关键点：
  - Tab：最新任务/任务模板/任务回收站
  - 过滤：全部/我创建的
  - 搜索 + 创建任务 + 批量删除 + 刷新
  - 表格列较多（任务类型、就绪状态、运行时长、优先级、回收策略等）
- 当前原型状态：
  - 主体结构已具备
  - 任务模板/回收站交互深度不足（仅结构，无场景数据）
- 差异优先级：P1

### C. 模型微调
- 线上关键点：
  - 列表 + 状态 + 进度 + 操作入口
- 当前原型状态：
  - 基础列表有
  - 操作与筛选项较少
- 差异优先级：P2

### D. 推理服务
- 线上关键点：
  - 服务列表、QPS、延迟、状态
  - 服务创建入口与管理操作
- 当前原型状态：
  - 关键字段已覆盖
  - 操作项较少（伸缩、下线、详情等缺失）
- 差异优先级：P2

### E. 开发机
- 线上关键点：
  - 用户筛选
  - 创建开发机
  - 操作：保存镜像、删除
- 当前原型状态：
  - 主流程已覆盖
  - 细节字段和文案可继续贴近线上
- 差异优先级：P2

### F. 模型仓库
- 线上关键点：
  - 模型列表、来源、版本、状态
  - 上传/导入入口
- 当前原型状态：

- 差异优先级：P2

### G. 模型服务市场
