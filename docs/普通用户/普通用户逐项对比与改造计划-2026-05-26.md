- 参考平台：`https://aicloud.infrawaves.com/`

## 三、本地代码对比结论（需要改动项）

### P0（必须对齐）

1. 我的资源页：补“筛选/重置”交互区，表格字段与线上文案完全一致。
2. 任务监控页：补全“近一小时/近三小时/近六小时/近一天/近三天”快捷时间按钮；补“报表下载”交互状态；卡片文案对齐。
3. API KEY 页：列表行操作补“编辑/禁用”双入口与状态切换说明；统计卡文案对齐“最近 30 天调用”。
4. 账单页：补“筛选/重置”交互（线上存在）；明细区字段文案对齐。

### P1（高优先级）

1. 模型仓库：补 `预置模型/自定义模型` Tab，补 `部署/微调/导入模型` 按钮组。
2. 推理服务：表头补 `推理框架/就绪状态/当前副本数/实例数量`。
3. 模型微调：补齐与线上一致的列表列顺序与文案。
4. 用户管理：补“支付方式绑定”按钮的“选择用户后可用”状态联动。

### P2（可后续优化）

1. 各页面“筛选/重置”按钮统一布局与视觉样式。
2. 空状态文案、分页区文案与线上统一。
3. 卡片数值格式（千分位、单位）精修。
